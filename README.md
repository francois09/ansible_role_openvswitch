OpenVSwitch
===========

Simple role to install and configure OpenVSwitch on a 2 hosts with VMs intercommunication environment.

Role Variables
--------------

By default the role didn't install OpenVSwitch nor configure

```
ovs__install: False
ovs__configure: False

```
